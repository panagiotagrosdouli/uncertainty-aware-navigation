**Uncertainty-Aware Navigation is a working synthetic research demo for risk-aware mobile-robot planning under uncertain occupancy-grid maps.**

> Research question: **How can an autonomous robot navigate safely when its map, perception, and environment model are uncertain?**

The repository now contains runnable code that generates metrics, figures, `demo.gif`, and `demo.mp4`. All generated outputs and benchmark tables are clearly labeled **Synthetic Demo**. They are useful for reproducible software validation, not for real-world robot-performance claims.

## Planning objective

For a path `π`, the prototype evaluates:

```text
J(π) = Σ [ path_cost + λ_u uncertainty + λ_r risk - λ_rec recoverability ]
```

Shortest-path planning is insufficient under map uncertainty because geometrically short routes may pass through unknown cells, narrow passages, low-confidence regions, or states with poor escape options.

## Limitations

This is a synthetic grid-world research prototype. It does not claim state-of-the-art results, hardware safety, formal verification, or deployment readiness. Recoverability is intentionally marked as a prototype scaffold.
